## ✨ Features

- 🔐 **Role-based Access**  
  - 👨‍🏫 **Admin** → Manage Subjects, Chapters & Quizzes  
  - 👩‍🎓 **User** → Attempt quizzes & view results  

- 🖥 **Modern UI (Vue.js)**  
  - Smooth navigation  
  - Mobile responsive design  
  - SPA experience  

- ⚡ **Fast & Secure Backend (Flask)**  
  - REST APIs for quiz & results  
  - Secure authentication  
  - Lightweight **SQLite** storage  

- 📊 **Real-Time Results** with instant evaluation
